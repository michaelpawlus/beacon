# Beacon

AI-powered job search platform that curates AI-first companies with evidence-backed scoring, automated career page scanning, job relevance scoring, and LLM-powered application materials generation.

**Built with:** Python, SQLite, Typer, Rich, httpx, BeautifulSoup4, Anthropic Claude API

## Outcomes

- 38 companies scored across 5 AI-first dimensions
- Automated career page scanning with relevance scoring
- LLM-powered resume and cover letter generation
- 274 passing tests across 3 phases
